大二暑假开始，也就意味着大二过去了，大三要来了。下个学期的交换生活一结束，就得开始愁心实习了。所以暑假到下学期结束，是用来专攻一门，提高自己竞争力的最好时光。

其实以后的选择还有很多，身边的每个人在不知不觉的时候就渐渐拥有了清晰的方向，可是出国深造对于自己来说确实是一个不小的诱惑，心里还是想要遇见更多未知的可能。但是前端貌似没有特别对口的专业，而且出国深造给家里带来的负担我也无法估计，这种可能估计就此搁浅了。但我相信“念念不忘，必有回响”，心中所想的终究会以某种方式实现。而现在，既然三年的计划就是直接就业，那么毫无疑问，实践能力相对而言就十分重要了。

今天辛苦了一个下午，做的 demo 虽然比较简单，但是心里还是十分开心的。而不足也是很明显的，首先模仿的痕迹连自己都觉得相当严重，然后 javascript 的 DOM 选择也非常不熟悉，所以 toggle 效果也没有实现。

终究还是缺乏系统的学习，想着这一个月还是先熟悉 html + css， 个人感觉javascript 的水好深呀，还是先把静态页面做到足够熟悉，下一步再好好学习 javascript 吧。接下来好好记录每一个 demo ，一步步慢慢走，总会达到我期望的高度。
